# Pokédex - Prova Pokémon

Um projeto de gerenciamento de Pokédex desenvolvido em **Dart**, com visualização dos resultados em **HTML**.

## 📁 Estrutura do Projeto

- **main.dart** - Código principal em Dart com lógica de Pokédex
- **resultado.txt** - Resultados das operações em formato texto
- **resultado.html** - Visualização HTML com tema Pokémon

## 🎯 Funcionalidades

✅ Gerenciamento de Pokémons  
✅ Captura e evolução de Pokémons  
✅ Sistema de batalhas  
✅ Filtros avançados  
✅ Estatísticas gerais  
✅ Marca de favoritos  

## 📊 Questões Implementadas

1. **Q1** - Pokémons Iniciais
2. **Q2** - Aumento de Nível
3. **Q3** - Operações de Pokédex
4. **Q4** - Listagem Completa
5. **Q5** - Filtros de Pokémons
6. **Q6** - Pokémons após Batalha e Captura
7. **Q7-Q8** - Histórico de Batalhas
8. **Q9** - Status de Captura e Favoritos
9. **Q10** - Análises Avançadas
10. **Q11** - Estatísticas Gerais
11. **Q12** - Evoluções e Batalhas Finais

## 🚀 Como Usar

### Executar o código Dart
```bash
dart main.dart
```

### Visualizar os resultados
Abra o arquivo `resultado.html` em um navegador web:
- Duplo clique em `resultado.html`
- Ou arraste para o navegador

- **Dart** - Lógica do programa
