# Pokédex - Prova Pokémon

Um projeto de gerenciamento de Pokédex desenvolvido em **Dart**, com visualização dos resultados em **HTML**.

## 📁 Estrutura do Projeto

- **main.dart** - Código principal em Dart com lógica de Pokédex
- **resultado.txt** - Resultados das operações em formato texto
- **resultado.html** - Visualização HTML com tema Pokémon

## 🎯 Funcionalidades

✅ Gerenciamento de Pokémons  
✅ Captura e evolução de Pokémons  
✅ Sistema de batalhas  
✅ Filtros avançados  
✅ Estatísticas gerais  
✅ Marca de favoritos  

## 🎨 Tecnologias

- **Dart** - Lógica do programa
- **HTML5** - Estrutura da visualização
- **CSS3** - Estilo com tema Pokémon
- **JavaScript** - Interatividade
